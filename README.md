# AI-Powered Communication Automator
This repository contains autonomous AI agent workflows built with **n8n** and **Google Gemini LLM**.

## Features
- **Email Classifier:** Automatically categorizes Gmail inquiries into 'Orders' and 'General Inquiries'.
- **Smart Drafting:** Uses Gemini AI to generate context-aware email replies saved directly to Gmail Drafts.
- **WhatsApp Integration:** (In Development) Real-time automated responses for business WhatsApp.

## How to Use
1. Install n8n.
2. Import the `gmail-ai-classifier.json` file into your n8n workflow editor.
3. Configure your Gmail and Google Gemini credentials.
